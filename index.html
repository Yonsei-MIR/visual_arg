What is Visual Argument?

  • Images to persuade, or communicate message using
    visual elements, color, text, and frame.
  • Has ability to convey complex messages quickly and emotionally.
  • Used in Advertising, cartoons.
  • Requires selective vision comprehend complex messages.

Why is it challenging?

  • Understanding visual arguments is hard even for the humans.
  • Requires selective vision: focusing on specific visual cures within an image

Our Solution:

  • We design reasoning process in interpreting.
  • VisArgs: A Benchmark for evaluation of understanding visual arguments.

A. Data Collection

    1. We collected around 1,600 advertisement images and 1,600 cartoons from a dedicated website.
    2. The Cooperation between human workers and a model (GPT-4-o) captured necessary visual cues.
    3. Given the visual premises and image, the model annotated three argumentation structure: commonsense premises, conclusions, and argument trees.
    4. Experienced human workers then adjust the annotations for greater accuracy.
    5. The final dataset consists of 1,611 images, including 914 advertisement images and 697 cartoons.     







B. Tasks and Experiments

Task 1: Localization of Premises

  • Specification:
    Given a visual argument, ground the target visual cues.
    Both closed and open set groundings were defined.
  • Purpose:
    Determine if difficulties in understanding visual arguments originate from basic visual perception.
  • Findings:
    Current models are effective in matching descriptions and correct regions. But suffer in semantic open-set grounding.

Task 2: Identification of Premises

  • Specification:
    Given a visual argument and its intermediate conclusion, select a visual premise that leads to the target conclusion.
  • Purpose:
    Assess the selective attention capabilities.
  • Findings:
    Models struggle to infer which visual cues support the argument.

Task 3: Deduction of Conclusion

  • Specification:
    Given a subset of inputs among the image, visual premises, commonsense premises, and the reasoning tree, generate the conclusion of an argument
  • Purpose:
    Comprehensive ability evaluation.
  • Findings:
    Selective attention to visual premises is a bottleneck.

Conclusion

  • Selective vision is a critical bottleneck for visual reasoning in current machines.
